# Enhanced Service Configurator - Refined Specification
## Code by Leon | "Get In Touch" Interactive Experience with Reinforcing Feedback

---

## Page Context
**URL**: `/get-started.html`  
**Purpose**: Replace traditional contact forms with a concierge-style intake that qualifies leads, gathers sufficient data, and delivers personalized recommendations with positive reinforcement throughout the journey.  
**Entry Points**: "GET IN TOUCH" CTA button, "Book a free 20-minute call" hero CTA

**Key Design Principles**:
- **Balanced Length**: 7-8 steps total (not too short to miss data, not too long to cause drop-off)
- **Reinforcing Feedback**: Encouragement messages after key milestones
- **Progress Visibility**: Clear progress indicators showing value
- **Positive Psychology**: Celebrate small wins, acknowledge honesty, build momentum

## Progress Indicator Design

### Visual Progress Bar
- **Position**: Top center, fixed
- **Style**: Dashed line (similar to Intch)
  - **Completed segments**: Solid black/grey dashes
  - **Current segment**: Highlighted with brand orange
  - **Upcoming segments**: Light grey dashes

### Step Counter
- **Format**: "Step X of 7" (or 8 if including contact step)
- **Position**: Below progress bar, centered
- **Style**: Small, subtle text

### Progress Dots (Alternative/Additional)
- **Active**: Filled orange (`#D9751A`)
- **Completed**: Filled with checkmark icon
- **Upcoming**: Outlined grey

## Feedback Screen Design Specifications

### Visual Elements
1. **3D Avatar Illustrations**
   - Different avatar for each feedback screen
   - Hexagonal background shape (light blue/purple gradient)
   - Friendly, approachable expressions
   - Consistent with brand aesthetic

2. **Color Scheme**
   - Light mode: Soft blues, purples, warm oranges
   - Dark mode: Deeper blues, warm accents
   - Maintain brand orange for CTAs

3. **Typography**
   - **Headline**: Large, bold, black (light mode) / white (dark mode)
   - **Message**: Medium size, grey text with bold keywords
   - **CTA Button**: Black background, white text

### Animation
- **Entry**: Fade in + subtle scale (0.95 â†’ 1.0)
- **Exit**: Fade out + slide transition to next step
- **Duration**: 300-400ms for smooth feel

## Step Flow Summary

**Total Steps**: 7-8 steps
- **Data Collection Steps**: 5 steps (Business Identity, Digital Presence, Goal, Timeline, Budget)
- **Additional Data Step**: 1 step (Additional Needs - multi-select)
- **Contact Step**: 1 step (Name, Email, Phone, Business Name)
- **Feedback Screens**: 3 screens (after steps 1, 3, 5)
- **Final Recommendation**: 1 screen

**Total User Interactions**: ~10-11 screens (feels substantial but not overwhelming)

## Psychology & Conversion Optimization

### Positive Reinforcement Strategy
1. **Early Win** (After Step 1): Celebrate their business type selection
2. **Mid-Journey Boost** (After Step 3): Validate their goal choice with data
3. **Honesty Reward** (After Step 5): Thank them for being transparent about budget
4. **Final Celebration** (Step 8): "All set!" with personalized recommendation

### Momentum Building
- Each feedback screen reinforces they're making progress
- Social proof integrated naturally ("47+ Kenyan businesses")
- Data-backed statements ("3x better results")
- Clear value proposition at each stage

### Trust Building
- Acknowledge honesty (especially around budget)
- No pressure messaging
- Transparent about process
- Free, no-obligation offer

## Technical Implementation Notes

1. **State Management**: 
   - Save progress in `localStorage` (resume if user refreshes)
   - Include feedback screen completion status

2. **Analytics Tracking**:
   - Track drop-off at each step
   - Track feedback screen engagement
   - Track which options are most selected
   - Track conversion to final CTA

3. **Form Validation**:
   - Email format validation
   - Phone number format (optional but helpful)
   - Required field indicators

4. **Email Integration**:
   - Send recommendation email via backend/API
   - Include all collected data for follow-up
   - Template with personalized recommendation

5. **Booking Integration**:
   - Calendly link with pre-filled information
   - WhatsApp link with pre-filled message template

6. **Accessibility**:
   - Full keyboard navigation
   - ARIA labels for all interactive elements
   - Screen reader friendly feedback messages
   - Focus management between steps
